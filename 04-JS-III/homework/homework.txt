	* Arrays

ARRAYS:
        Un array es una lista de elementos, que puede contener muchos elementos o pocos elementos.
        Los arrays pueden contener cualquier tipo de datos (numeros, strings, booleanos, funciones, arrays, etc.)
        La sintaxis correcta para crear un array es:
        let persona = ["Victoria", 25, "Traductora Pública de inglés", true];
